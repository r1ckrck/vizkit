# fal-image custom preset

Copy this file to your project (any name, any path), edit values, then reference it in your request:

> "using @./brand.preset.md make me a hero image for the homepage"

All fields are optional. Leave empty fields as `—` — Claude infers them from your request.

The structure below matches the Cinema domain preset. For other domains (Product, Portrait, Editorial, UI, Logo, Landscape, Abstract, Infographic), copy the corresponding preset from `~/.claude/skills/fal-image/presets/<domain>.preset.md` instead — each has its own domain-specific fields.
